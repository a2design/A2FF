A2 Framework
============

Leia-me versão 0.1

Estrutura das pastas
--------------------

bck - pasta de backup. deve ser deletada quando em novo projeto.

css - pasta com os arquivos css do projeto, exceto snippets.

demo - pasta com arquivos php para demonstração.

img - pasta com arquivos multimídia.

in - pasta para includes php.

js - pasta para arquivos javascript, exceto snippets.

snippets - pasta que habilitará a funcionalidade homônima no Aptana. Contém arquivos html, css e js.
